# Support TurkiyeAPI

TurkiyeAPI is free to use, public, and open source. If the API helps your project, you can support the hosting and maintenance costs.

Support is optional. The API remains available without an API key, paid plan, or donation requirement.

## How Donations Are Used

Donations help cover:

- API hosting and server costs.
- Maintenance work for the public API and documentation.
- Data refreshes, fixes, and future improvements.

## Sponsorships

If you want to sponsor the project in another way, open a discussion or contact the maintainer through the [GitHub repository](https://github.com/ubeydeozdmr/turkiye-api).

Sponsorship does not grant special access to private data, bypass rate limits, or change the public nature of the API.
