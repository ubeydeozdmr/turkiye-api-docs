# TurkiyeAPI'yi Destekleyin

TurkiyeAPI ücretsiz, herkese açık ve açık kaynaklıdır. API projenize yardımcı oluyorsa, sunucu ve bakım maliyetlerini karşılamak için destek olabilirsiniz.

Destek tamamen isteğe bağlıdır. API; API anahtarı, ücretli plan veya bağış şartı olmadan kullanılmaya devam eder.

## Bağışlar Nasıl Kullanılır?

Bağışlar şu kalemlere destek olur:

- API hosting ve sunucu maliyetleri.
- Public API ve dokümantasyon bakımı.
- Veri güncellemeleri, düzeltmeler ve gelecek iyileştirmeler.
